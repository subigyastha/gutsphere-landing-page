Phase # Cohort One-line (from positioning doc)

2Early multiple symptoms Pattern emerging, nobody connecting it, patient becomes the integrator

4Undiagnosed Sick but without answers — dismissed, gaslit, still searching
5Misdiagnosed Has a label, but it doesn’t fit — treatment isn’t working
